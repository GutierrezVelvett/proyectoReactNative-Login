# Gestión de Inventario Cappdi

## Descripción
Esta aplicación móvil desarrollada en React Native permite gestionar el inventario de manera eficiente. Incluye funcionalidades de autenticación para asegurar el acceso seguro a los datos del inventario.

## Características
- **Autenticación de usuarios**: Pantallas de login y registro para controlar el acceso.
- **Gestión de inventario**: Funcionalidades para agregar, editar y eliminar productos del inventario.
- **Interfaz intuitiva**: Diseñada con componentes reutilizables para una experiencia de usuario fluida.
- **Estilos personalizados**: Utiliza estilos modulares para mantener consistencia en la aplicación.
